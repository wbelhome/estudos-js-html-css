Posicionado os elementos em tela

A propriedade position ajuda a manipular o elemento na sua pagina

  • position:static, é o padrão do css
  • position:relative, destrava propriedade de top, right, left e bottom, que posiciona o elemento para as direções determinadas
  • position:absolute, destrava propriedade de top, right, left e bottom, porém se posiciona em relação a tela
  • position:fixed
  • position:sticky
