Imagem object-fit

object-fit: fill ; (padrão)
FILL faz a imagem pegar os valores padrões da tag no css, distorcendo a imagem

CONTAIN mantem o tamanho da imagem sem precisar distorcer ela para caber no container
object-fit: contain ;

COVER Cobre todo o espaço, ultrapassando o limite do container
object-fit: cover ;

NONE não utiliza nem um padrão estabelecido e sim o tamanho original da imagem
object-fit: none ;

SCALE-DOWN escolhe a menor imagem entre NONE e CONTAIN
object-fit: scale-down ;
