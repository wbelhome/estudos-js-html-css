Pseudo-classes: VALID, INVALID, IN-RANGE e OUT-RANGE

Identificado por :palavra-chave

  • invalid: estiliza quando o campo estiver invalido
  • valid: estiliza quando o campo estiver valido
  • in-range: estiliza quando estiver entre os valores estabelecidos no MIN e MAX
  • out-range: estiliza quando estiver entre os valores estabelecidos no MIN e MAX
