Bordas

Podemos definir as configurações das nossas bordas através das propriedades:

  • border-image-source
  • border-image-width
  • border-image-repeat
  • border-image-outset
  • border-image-slice

EXEMPLO:

border-image-SOURCE


border-image-source linear-gradient


border-image-slice == fatiado


border-image-repeat: space


border-image-repeat: repeat;


border-image-repeat: stretch;


border-image-outset: valor;


border-image-outset: com varios;


border-image-outset: stretch;
